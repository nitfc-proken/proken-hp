## 環境構築

- Node.jsを公式サイトからインストール
- GitHub Desktopをインストール、このリポジトリをclone
```bash
#GitHub Desktopでcloneした人はgit cloneは必要なし
$ git clone https://github.com/nitfc-festival-system-team/2023hp
$ cd 2023hp
$ npm install
$ npm run dev
```
- http://localhost:3000/ にアクセスし、結果を確認する
