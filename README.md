## 環境構築

- Node.js を公式サイトからインストール
- GitHub Desktop をインストール、このリポジトリを clone

```bash
#GitHub Desktopでcloneした人はgit cloneは必要なし
$ git clone https://github.com/nitfc-proken/proken-hp && cd proken-hp
$ npm install
$ npm run dev
```

- http://localhost:3000/ にアクセスし、結果を確認する
